OCULI

About

The game is set in an old mansion, embracing classic horror and puzzle game tropes. It features iconic horror elements like shifting rooms, vanishing doors, and eerie apparitions. The player must solve three puzzles to advance the story while the mansion changes and new enemies emerge, revealing the player's position to a pursuing monster.

Project info

Role: Gameplay Programmer

Team size: 8

TimeFrame: 3 months

Engine: Unreal Engine 5 (blueprint)
